# Commit and Push Checklist

1. `git status --short` is reviewed.
2. Branch is `main`.
3. `git fetch origin` completed.
4. Local `main` is not behind remote (or rebase completed).
5. No secrets or large binaries are staged accidentally.
6. Commit message is meaningful and scoped.
7. `git push origin main` succeeds.
8. Final report includes commit hash and file summary.
