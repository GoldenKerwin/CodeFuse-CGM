# Common Heavy File Patterns

Use these as a baseline and adapt to repository specifics.

## Data and Generated Outputs
- `data/**`
- `datasets/**`
- `outputs/**`
- `artifacts/**`
- `checkpoints/**`

## Model and Weight Files
- `*.pt`
- `*.pth`
- `*.ckpt`
- `*.onnx`
- `*.bin`
- `*.safetensors`
- `*.h5`

## Caches and Logs
- `.cache/`
- `__pycache__/`
- `.pytest_cache/`
- `*.log`
- `wandb/`
- `runs/`
- `tb/`

## Temporary and OS Files
- `tmp/`
- `temp/`
- `.DS_Store`
- `Thumbs.db`
