# skills

仓库内维护的 Codex 自定义技能（源码版本）。

## 当前技能
- `project-readme-maintainer/`: 生成/更新结构化 README
- `gitignore-large-file-guard/`: `.gitignore` 大文件防护与扫描
- `git-main-commit-pusher/`: 提交并推送 `main` 的流程化检查

## 说明
- 这些目录用于版本化管理技能定义
- 真正被 Codex 自动发现的安装位置是 `$CODEX_HOME/skills`（当前环境为 `/root/.codex/skills`）
